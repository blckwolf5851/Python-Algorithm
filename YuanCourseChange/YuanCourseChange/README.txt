This project change the timetable automatically base on user's need. Since there are different courses available at different period. To allow the students to have the course he/she want. The time table needs to be rearranged in a way that every course are available at that specific period.
Example operation:
1. change course 1 to second semester
2. insert course 3 to first semester
3. switch course 1 to course 6
etc...
